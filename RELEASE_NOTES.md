EzekoConcept OBS Countdown Timer — v1.0.0

Release date: 2025-11-09

Highlights
- Initial public release of the EzekoConcept OBS Countdown Timer.
- Two-page static timer: large display (`index.html`) and settings/control panel (`settings/index.html`).
- localStorage-based communication between pages.
- Light/dark theming on the display; settings UI forces readable black text.
- OBS-friendly: use "Local file" in Browser Source or `file:///` paths for docks.

Included files
- index.html, settings/index.html, js/, css/, screenshots/, README.md, LICENSE
